Что изменилось в интерфейсе
🎨 Визуальный стиль
Цветовая система Заменил нейтральный серо-белый интерфейс на брендовую палитру Ozon: синий #005BFF как основной акцент, оранжевый #FF6A00 для realFBS, зелёный #00A859 для FBS. Каждый канал теперь имеет свой устойчивый цвет по всему интерфейсу.

Шрифт Заменил системный шрифт на Golos Text — современная гарнитура с отличной поддержкой кириллицы, разработанная специально для русского языка.

Карточки вместо плоских блоков Все секции оформлены как белые карточки с мягкой тенью, скруглёнными углами и тонкой рамкой. Убран «корпоративный» вид с резкими границами.

Цветовая акцентная полоска на KPI-плашках — каждая карточка имеет цветную полосу сверху, отражающую смысл метрики.

📊 Новые компоненты
KPI-плашки (сводка наверху) Добавлен ряд из 5 карточек: суммарная маржа FBO / FBS / realFBS, лучший канал, рентабельность к себестоимости. Все данные пересчитываются при изменении фильтра канала. Есть кнопка «Скрыть / Показать сводку» — плашки можно свернуть одним кликом.

Фильтр каналов Заменил отсутствующий фильтр на цветные pill-кнопки: Все / FBO / FBS / realFBS. При выборе канала фильтруется таблица и пересчитываются KPI.

Мини-бар-чарт маржи В каждой строке товара добавлены три маленьких столбика, визуально сравнивающих маржу по каналам. Лучший канал подсвечен ярче. Можно отключить через Tweaks.

✏️ Inline-редактирование таблицы
Поля цена, себестоимость, кол-во редактируются прямо в таблице — клик по ячейке превращает её в input. Подтверждение: Enter или клик вне поля. Отмена: Escape.

Логика read-only для Ozon-товаров Артикул, название и категория у импортированных товаров заблокированы — эти данные приходят из каталога Ozon и не должны меняться вручную. Иконка Oz рядом с артикулом и подсказки при наведении объясняют почему. У товаров добавленных вручную все поля редактируемы.

Легенда в тулбаре таблицы: «Oz — данные из Ozon, только чтение».

💰 Вкладка Финансы
Сводные KPI-карточки показывают суммарные доходы, расходы, чистый доход и количество операций за период.

Цветные теги типов операций: Продажа → зелёный, Комиссия → красный, Логистика → оранжевый, Возврат → фиолетовый.

Фильтр по типу операции + строка итого в подвале таблицы с цветом в зависимости от знака суммы.

📱 Мобильный вид
KPI-карточки переносятся в несколько строк через flex-wrap
Таблица горизонтально скроллируется с инерцией (-webkit-overflow-scrolling: touch)
Все кнопки ≥ 44px по высоте
Уменьшенные отступы и размер заголовка на экранах < 640px
⚙️ Tweaks-панель
Кнопка в тулбаре открывает панель настроек:

Акцентный цвет — меняет синий на любой
Тёмная шапка — переключает header в тёмный режим (#001A4D)
Мини-график маржи — показать/скрыть столбики в таблице
Плотность — обычная / компактная (меняет отступы контента)
📐 Ширина страницы
Убрано ограничение maxWidth: 1280px — интерфейс теперь использует всю доступную ширину экрана. Таблица с 12+ колонками перестала требовать горизонтальный скролл на стандартных мониторах.
